Sistema operativo: Es el software base que administra los recursos de hardware de la computadora (como memoria y procesador). 
Permite que el entorno de ejecución, como Node.js, interactúe con el equipo físico y ejecute la aplicación.

Node.js: Es un entorno de ejecución de JavaScript que permite correr este lenguaje de programación del lado del servidor. 
Se encarga de interpretar y ejecutar el código backend fuera del navegador web tradicional.

Aplicación server.js: Es el archivo de código fuente principal que contiene la lógica de nuestro servidor web.
 Define las rutas, cómo manejar las peticiones de los usuarios y qué respuestas debe enviar (como los datos JSON).

Puerto: Es una interfaz virtual o canal de comunicación específico que usa el servidor para escuchar conexiones. 
Permite que el tráfico de internet identifique exactamente a qué aplicación debe dirigirse dentro del equipo.

Logs: Son registros de texto automáticos donde se guarda el historial de eventos, errores o actividades del servidor. 
Sirven para monitorear el comportamiento de la aplicación y detectar fallas en tiempo real.
